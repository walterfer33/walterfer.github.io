    • Instalación de node.js

    Comando para actualizar la terminal

    
		sudo yum update

    Instale el administrador de versiones de nodo (nvm) escribiendo lo siguiente en la línea de comando.

    
		curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash

    Usaremos nvm para instalar Node.js porque nvm puede instalar múltiples versiones de Node.js y permitirle cambiar entre ellas. Active nvm escribiendo lo siguiente en la línea de comando.

    
		. ~/.nvm/nvm.sh

    Use nvm para instalar la última versión de Node.js escribiendo lo siguiente en la línea de comando.

    
		nvm install node

    Pruebe que Node.js esté instalado y funcionando correctamente escribiendo lo siguiente en la línea de comando.

    
		node -e "console.log('Running Node.js ' + process.version)"

    Comando para agregar el repositorio de node.js

    
		curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
